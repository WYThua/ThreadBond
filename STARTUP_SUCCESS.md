## 🛠️ 技术架构

### 后端技术栈
- **框架**: Node.js + Express + TypeScript
- **数据库**: MySQL + Prisma ORM
- **缓存**: Redis
- **认证**: JWT + bcrypt
- **验证**: Joi 数据验证
- **安全**: 频率限制 + 输入过滤

### 前端技术栈
- **框架**: Vue.js 2.x
- **UI库**: Vant (移动端)
- **状态管理**: Vuex
- **路由**: Vue Router
- **样式**: SCSS + 响应式设计

### 部署方案
- **容器化**: Docker + Docker Compose
- **开发环境**: 热重载 + 实时编译
- **数据持久化**: Docker 卷挂载
